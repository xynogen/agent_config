# TLDR Directive
## Below are what agent MUST do:
- **AUTO-RUN**: Proactively execute terminal commands and tool calls needed to fulfill this directive without waiting for confirmation unless explicit user input is required.
- **SCOPE**: Tech-only explanation. No pleasantries or conversational filler.
- **WORKFLOWS**: Identify relevant workflows and their purposes.
- **DATA**: Key data points, configurations, and assumptions.
- **CONFIDENCE**: State confidence levels for uncertain information.
- **DELIVER**: Maximum density, minimum length. Bullet points preferred.
