# Explain Directive
## Below are what agent MUST do:
- **AUTO-RUN**: Proactively execute terminal commands and tool calls needed to fulfill this directive without waiting for confirmation unless explicit user input is required.
- **CONTEXT**: Clearly identify the target code, file, or concept to explain.
- **DECONSTRUCT**: Break down the logic into understandable components and flows.
- **RATIONALE**: Explain the "why" behind design decisions and implementation choices.
- **TRACE**: Map data flow, call chains, and dependencies through the codebase.
- **REPORT**: Provide clear, technical explanation with direct file references and examples.
