# Suggest Directive
## Below are what agent MUST do:
- **AUTO-RUN**: Proactively execute terminal commands and tool calls needed to fulfill this directive without waiting for confirmation unless explicit user input is required.
- **IDEATE**: Brainstorm improvements across Developer Experience (DX), User Experience (UX), Performance, and Maintainability.
- **ANALYZE**: Evaluate each suggestion against current constraints and best practices.
- **RANK**: Prioritize by Impact vs Effort ratio and business value.
- **PROPOSE**: Present actionable recommendations with implementation guidance.
- **REPORT**: Clear categorization with rationale and priority levels.
